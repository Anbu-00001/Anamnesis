# Agent memory — snapshot

A point-in-time copy of the Claude Code **agent memory** for this project, captured
so the *why* survives a re-clone, a device change, or a Claude account switch — not
just the local `~/.claude/` store (which is machine-bound and lost if the device is).

- **Captured:** 2026-06-09
- **Live source (canonical, not version-controlled):**
  `~/.claude/projects/-home-anbu-26-class-PlayGround/memory/`
- **What's here:** `MEMORY.md` (the index) plus one file per memory
  (`project_*`, `reference_*`).

This is a *backup snapshot*, not the live store — it can go stale. The running
agent still reads/writes the `~/.claude/...` copy each session. Re-run the copy to
refresh. For Anamnesis specifically, the distilled rationale also lives, in prose,
in the repo-root `CLAUDE.md` ("Design rationale & decisions"); this folder keeps the
raw, undistilled originals alongside it.
